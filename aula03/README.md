# Aula 3 — Diagramas, Dataclasses e SRP (assíncrona)

Sistema da aula: **Academia FitPará** — um sistema de check-in onde **uma única
classe faz tudo** (regra de negócio + tela + notificação) e os dados são
dicionários soltos. Sua missão é começar a separar isso com o **SRP**.

## O que tem aqui
```
aula03/
├── academia.py        ← o sistema v1.0 para ESTUDAR (roda; é uma classe que faz tudo)
├── docs/              ← visão, requisitos (RNF) e projeto
└── template/          ← onde você TRABALHA e ENTREGA (diagrama, análise e código)
```

## Como começar
1. Leia o **roteiro da aula** no SIGAA (tem todo o conteúdo e o passo a passo).
2. Rode o sistema atual para conhecê-lo:
   ```bash
   cd aula03
   python academia.py
   ```
3. Faça a atividade dentro de `template/` (veja `template/README.md`).
4. `git push` e cole o **link do seu repositório** no SIGAA.

> Esta é uma aula **assíncrona**: você faz em casa, no seu tempo. A entrega da
> tarefa no prazo conta como sua **presença** no encontro.
